# 2D Racing Game

2D Racing Game은 Python과 pygame을 사용하여 제작 중인  
**로컬 멀티플레이 기반 2D 레이싱 게임**

이 프로젝트는 단기간(4일) 내에 완성 가능한 구조를 목표로 하며,  
 **두 컴퓨터 간 대결 레이싱 게임**이 핵심 목표이다.

---

## 주요 특징
- pygame 기반 2D 레이싱
- 고정 화면의 작은 트랙 (스크롤 없음)
- 벽 충돌 및 슬라이딩 이동 처리
- 체크포인트 기반 승리 조건 (3개 순서 통과)
- 로컬 2P → 네트워크 대전으로 확장 가능한 구조
- 완료주의(Complete-first) 개발 전략

---

## 현재 구현 상태
- 자동차 가속 / 브레이크 / 회전
- 벽 충돌 및 벽을 따라 미끄러지는 슬라이딩 처리
- 체크포인트 순서 판정
- 2인 플레이를 위한 LAN 통신기능
- 2인 플레이 
- 카운트 다운
- 신호등
- 아이템(부스트)
- 버튼을 통한 이모지 표현 5가지
- 여러가지 맵
- 속도/가속도 조정
- 승패 판정 시 순위 디스플레이
- 게임 시작화면 이미지
- 차량 스타드 위치 통일
- 차 간의 충돌 구현
