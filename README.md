# 2D Racing Game

2D Racing Game은 Python과 pygame을 사용하여 제작 중인  
**로컬 멀티플레이 기반 2D 레이싱 게임**

이 프로젝트는 단기간(이틀) 내에 완성 가능한 구조를 목표로 하며,  
향후 소켓 통신을 통해 **두 컴퓨터 간 대결 레이싱 게임**으로 확장하는 것을 염두에 두고 있다.

---

## 주요 특징
- pygame 기반 2D 레이싱
- 고정 화면의 작은 트랙 (스크롤 없음)
- 벽 충돌 및 슬라이딩 이동 처리
- 체크포인트 기반 승리 조건 (3개 순서 통과)
- 로컬 2P → 네트워크 대전으로 확장 가능한 구조
- 완료주의(Complete-first) 개발 전략

---

## 현재 구현 상태
- 자동차 가속 / 브레이크 / 회전
- 벽 충돌 및 벽을 따라 미끄러지는 슬라이딩 처리
- 이미지 스타일을 참고한 곡선형 트랙 구성 (Rect 기반)
- 체크포인트 순서 판정
- 싱글 플레이 완료
